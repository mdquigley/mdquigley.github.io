Portfolio: For application to the NYU Steinhardt Master of Music in Music Technology program.

Mirror Music: Final project for Advanced Max/MSP/Jitter Programming with Professor Dafna Naphtali.

Summary: An interactive synthesizer that generates random pitches within a user specified key, triggered by motion, and characterized in range and timbre by color.

FUNCH: Final project for Computer Music Synthesis with Professor Peter McCulloch.

Summary: A Max/MSP patch intended as a performance application with opportunities for collaboration between patch operator and live performer or prerecorded sound sources.

Visual Feedback In A Laptop Instrument

Summary: Final project for Psychology of Music with Professor Mary Farbood. This experiment tested the hypothesis that, in a video camera controlled software instrument, the presentation of visual feedback will aid in the user's understanding of musical mappings to visual space. The "visual feedback" was based off of a system similar to that used in "Mirror Music" and the completed project acted as the preliminary research stage of my masters thesis.

Fundamentals of Digital Signal Theory

Summary: Final paper for Fundamentals of Digital Signal Theory with Dr. Agnieszka Roginska. Reports on the planning and implementation of a ring modulation and short time fourier transform function in MATLAB. Source code is available here.

Zone Out

Written as the final project for Java Music Systems with Professor Nick Didkovsky, Zone Out is a java program that creates generative ambient music for three voices, written with JMSL and JSyn. Final paper and source code are available here. Below is an excerpt.

Okay

Written as the final project for Advanced Computer Music Composition with Professor Tae Hong Park. Sound sources are from electronic synthesis, processed and edited digitally as a tape piece.

Washer

Washer is a touring & recording project that I share with one other member. We have released six albums (LPs / EPs / 7"s) for which I've overseen the songwriting, production, and graphic design. Below is a recent full length album, All Aboard.

Disquiet Junto

Some contributions to the Disquiet Junto project, a series of fast-turnaround assignments to compose, record, and share new music using constraints as a springboard for creativity.
